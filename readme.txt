Adds Interactive Calendar in Publish Section of Post, Page & Custom Post.

== Description ==

Publish Date DatePicker adds interactive calendar in publish section of post, page & custom post.

It makes adding and changing date easier by selecting it from interactive calendar using mouse.

== Installation ==

* Install the plugin from the 'Plugins' section in your dashboard (Go to `Plugins > Add New > Search` and search for Publish Date DatePicker).
* Alternatively, you can [download](http://downloads.wordpress.org/plugin/publish-date-datepicker.zip "Download Publish Date DatePicker") the plugin from the repository. Unzip it and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the 'Plugins' section.

== Frequently Asked Questions ==

= Does it cause to make site slow? =

No. It only loads tiny css,image and js file in edit screen of post, page and custom post.
